# libntrwifi
Drop-in replacement for NitroWiFi that targets Nintendo DS, PC, and other platforms.

libntrwifi makes it easier to port Nintendo DS games to PC and other platforms by handling the common network stack used by Nintendo DS games.

## Available Build Targets
Select the build target with the 'build_target' meson option.
 * arm (Nintendo DS hardware)
 * linux (x86_64 Linux PC)
 * win64 (x86_64 Windows PC)
 * nx (Switch via libnx)
